#Http Request Method
You can think fo this as the verb that tells the server what actiooon to perform in a resource.

##GET request

- **GET** request are used to retrieve a resource, and most link are GETs.
- The response form a GET request can be anything, but if it's HTML and that HTML reference other resources, your browser will automatically request those referenced resources. A pure HTTP tool will not.
- There are size and security limitations to using GET request
